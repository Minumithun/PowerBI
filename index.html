üîó Data Connection Mastery

1

Launch Power BI Desktop

Open Power BI Desktop and start a new project

üìã Step-by-Step Instructions:

  1. Open Power BI Desktop on your computer
  2. If you see a welcome screen, click "Get data" or select "Blank report"
  3. You should now see the main Power BI interface with the ribbon menu at the top
  4. Notice the three main panels: Report view (center), Visualizations (right), Fields (right)
Pro Tip: Make sure you have the latest version of Power BI Desktop for the best experience.
2

Import HR Employees Data

Connect to the HR employees CSV file

üìã Step-by-Step Instructions:

  1. Click "Get Data" in the Home ribbon
  2. Select "Text/CSV" from the dropdown menu
  3. Browse to your downloaded "hr-employees.csv" file
  4. Click "Open" to see the data preview
  5. Review the data and click "Load" to import it
Expected Result: You should see "hr-employees" appear in the Fields panel on the right with all the employee columns listed.
3

Add Recruitment Metrics

Import the second data source

üìã Step-by-Step Instructions:

  1. Click "Get Data" again in the Home ribbon
  2. Select "Text/CSV"
  3. Choose "recruitment-metrics.csv" from your downloads
  4. Preview the data and click "Load"
Challenge: Can you identify which columns might be used to link these two datasets together?
4

Add Performance Data

Complete your data foundation

üìã Step-by-Step Instructions:

  1. Import "performance-data.csv" using the same process
  2. Verify all three tables appear in the Fields panel
  3. Expand each table to see the available columns
  4. Your data foundation is now complete!

üìä Visualization Mastery

Create stunning charts and visual elements

0/5 Steps Complete
1

Create Employee Count by Department

Build your first column chart

üìã Step-by-Step Instructions:

  1. Look at the Visualizations panel on the right
  2. Click on "Clustered Column Chart" (vertical bars icon)
  3. From the hr-employees table, drag "Department" to the Axis field
  4. Drag "Employee_ID" to the Values field (it will auto-count)
  5. Watch your first chart appear!
Magic Moment: Power BI automatically counts employees when you drag Employee_ID to Values - no formulas needed!
2

Add Gender Distribution Pie Chart

Show proportional data with a pie chart

üìã Step-by-Step Instructions:

  1. Click on empty space in your report canvas
  2. Select "Pie Chart" from the visualizations panel
  3. Drag "Gender" from hr-employees to the Legend field
  4. Drag "Employee_ID" to the Values field
  5. Resize the chart by dragging its corners
3

Create KPI Cards

Display key metrics prominently

üìã Step-by-Step Instructions:

  1. Select "Card" visualization from the panel
  2. Drag "Employee_ID" to the Fields area (shows total employees)
  3. Create another card for "Salary" (shows average salary)
  4. Position these cards at the top of your report
  5. Cards are perfect for showing key numbers!
4

Add Hiring Trend Line Chart

Show trends over time

üìã Step-by-Step Instructions:

  1. Select "Line Chart" from visualizations
  2. Drag "Hire_Date" from hr-employees to the Axis
  3. Drag "Employee_ID" to Values
  4. Power BI will show hiring trends over time
  5. You can change the date hierarchy if needed
5

Create Employee Details Table

Display detailed information

üìã Step-by-Step Instructions:

  1. Select "Table" visualization
  2. Add these fields: Employee_Name, Department, Position, Salary
  3. Resize the table to show the data clearly
  4. You can sort by clicking column headers

Apply Consistent Colors

Create a cohesive color scheme

üìã Step-by-Step Instructions:

  2. Look for the Format pane (paint roller icon) on the right
  3. Expand "Data colors" section

2

Add Titles and Labels

Make your visuals clear and informative

üìã Step-by-Step Instructions:

  1. Select each visual one at a time
  2. In the Format pane, find "Title" section
  3. Turn the title ON
  4. Change the title text to be descriptive (e.g., "Employees by Department")
  5. Adjust font size and color as needed
3

Arrange Your Layout

Organize visuals for maximum impact

üìã Step-by-Step Instructions:

  1. Drag visuals to reposition them
  2. Resize by dragging corners and edges
  3. Place KPI cards at the top in a row
  4. Arrange charts in a logical grid below
  5. Leave appropriate white space between elements

Final Polish

üìã Step-by-Step Instructions:

  1. Click on empty space in your report (deselect all visuals)
  2. In the Format pane, look for "Page background"
